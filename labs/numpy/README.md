# NumPy Labs

One minimal script per operation (run any file with `python3 labs/numpy/<topic>/<name>.py` from the repo root).

| Folder | Topics |
|--------|--------|
| `arithmetic/` | Element-wise `+`, `-`, `*`, `/`, `**`, `%`, `sqrt`, `exp`, `sin`, `log` |
| `aggregation/` | `sum`, `mean`, `median`, `std`, `var`, `min`, `max`, `argmin`, `argmax`, `percentile` |
| `shape/` | `reshape`, `transpose`, `flatten`, `concatenate`, `split`, `ravel`, `expand_dims`, `squeeze`, `vstack`, `hstack` |
| `linalg/` | `dot`, `det`, `eye`, `inv`, `matrix_rank`, `eigvals`, `solve`, `norm`, `trace`, `outer` |

Legacy entry: `basic_numpy.py` (tiny array + mean).
