# Machine Learning Labs (Basic)

Very simple starter labs for machine learning.

## Files

- `ml_01_load_data.py`: load a CSV and print shape/columns.
- `ml_02_train_test_split.py`: split features and target.
- `ml_03_linear_regression.py`: train a basic linear regression and print MAE.

## Run examples

```bash
python labs/ml/ml_01_load_data.py
python labs/ml/ml_02_train_test_split.py
python labs/ml/ml_03_linear_regression.py
```
