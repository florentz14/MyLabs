# Data Visualization Labs

Basic charting examples using the naming pattern `viz_<topic>_<level>.py`.

## Entry scripts

| File | Role |
|------|------|
| `basic_viz.py` | Line plot saved to **`data/gen/basic_viz.png`** (no display window) |
| `viz_line_basic.py` | Interactive line chart (`plt.show()`) |
| `viz_bar_people_basic.py` | Bar chart from **`people.csv`** |
