INSTRUCTIONS — QUESTIONS (use Pandas)
-------------------------------------
1. Load the dataset into a Pandas DataFrame. Display the first 5 rows.

2. Show how many missing values there are in each column.

3. Remove rows with missing values in the Email column.

4. Remove duplicate rows from the dataset.
   Data note: if two rows match on every field except "Id", use
   drop_duplicates with a subset that excludes "Id" so the same business
   listed twice is treated as a duplicate. If the instructor requires plain
   drop_duplicates() with no subset, only rows that are 100% identical
   (including Id) count as duplicates.

5. Sort the companies alphabetically by CompanyName.

6. Sort the companies by Industry.

7. Count how many companies belong to each industry using groupby.

8. Show all companies that belong to the Technology industry.

BONUS QUESTION (+5 extra points)

Create a new column called HasWebsite:
  • If the company has a website → value "Yes"
  • If the company does not have a website → value "No"
  (Treat empty strings and NaN in Website as "no website".)
