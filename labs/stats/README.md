# Statistics Labs (`labs/stats`)

Exercises with Python’s standard-library **`statistics`** module: descriptive measures on numeric lists, **without pandas**.

For tabular analysis with **pandas** (CSV, `describe`, `value_counts`), use **`labs/pandas/stats/people_summary.py`** and the **`labs/pandas/basics/`** track (e.g. `10_describe.py`, `34_statistics.py`).

## Requirements

- Python 3.10+ (recommended).
- Standard library only; no extra `pip install` needed for these scripts.

## How to run

From the repository root:

```bash
python3 labs/stats/basic_stats.py
```

## Scripts

| File | Role | Description |
| --- | --- | --- |
| `basic_stats.py` | Descriptive stats | Mean, median, and sample standard deviation with **`statistics.mean`**, **`median`**, **`stdev`**. |
